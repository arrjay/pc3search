Making M-Crew Captures
======================

Currently I do my captures for the PC3 with a Windows 10 x86 VM to run M-Crew, and Wireshark running on the host system (Linux) using the usbmon interface.

I am using libvirt/virt-manager with SPICE USB passthrough to connect the MN10.

Capture should be *possible* entirely inside the Windows system, but I've no real desire to try at this time.
